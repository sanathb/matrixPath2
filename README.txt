matrixPath
==========

Given a matrix m*n, calculate all possible paths from left top to right bottom. 
*Precondition: Element can be traversed only once and you can move up,down, left or right



Matrix 3*3

0   1   2
3   4   5
6   7   8



The paths are:
-->0-->1-->2-->5-->8

-->0-->1-->4-->5-->8

-->0-->3-->4-->5-->8

-->0-->1-->4-->7-->8

-->0-->3-->4-->7-->8

-->0-->3-->6-->7-->8

-->0-->3-->4-->1-->2-->5-->8

-->0-->3-->6-->7-->4-->5-->8

-->0-->1-->2-->5-->4-->7-->8

-->0-->1-->4-->3-->6-->7-->8

-->0-->3-->6-->7-->4-->1-->2-->5-->8

-->0-->1-->2-->5-->4-->3-->6-->7-->8
